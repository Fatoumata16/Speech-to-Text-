# README - Archive

Ce dossier conserve les versions archivees des notebooks.

## Fichiers

- `projetBambara_full_backup.ipynb`
  - version **lisible** et **recentree** sur le fine-tuning Whisper,
  - nettoyee pour supprimer les parties non liees (ex: sections Wav2Vec2).

- `projetBambara_full_backup_raw.ipynb`

## Pourquoi deux versions

- La version lisible permet une prise en main rapide.
- La version brute preserve l'historique complet des essais.
