# Guide des fichiers

Ce document decrit le role de chaque fichier principal pour faciliter la lecture du projet.

## Scripts applicatifs

- `streamlit.py` : interface principale de demo (capture audio, transcription, traduction, comparaison des sorties).
- `app.py` : version simplifiee/prototype de l'interface Streamlit.
- `TTS.py` : fonctions et experimentation autour de la synthese vocale.

## Experimentation ML


- `archive/projetBambara_full_backup.ipynb` : version lisible centree sur le fine-tuning Whisper.
- `archive/projetBambara_full_backup_raw.ipynb` : version historique brute (complete, non nettoyee).

## Documentation

- `README.md` : vue d'ensemble du projet, objectifs, resultats et captures.
- `archive/README.md` : conventions d'archivage et statut des notebooks.

## Dependances

- `requirements.txt` : bibliotheques Python necessaires pour executer les scripts.
